# Skill: Geoservico OGC

Use esta skill quando:
- A pergunta envolver WMS, WFS, WMTS, WCS ou OGC API
- Precisar interpretar GetCapabilities e operacoes suportadas
- Montar URLs HTTP com parametros corretos
- Resolver erros de versao, CRS, eixo e filtros OGC/CQL

Instrucoes:
- Usar especificacoes oficiais OGC como base conceitual
- Destacar diferencas de versao e impactos praticos
- Mostrar exemplos de URL completos e validos
- Explicar diagnostico de erro com causa e acao corretiva

Referencias condicionais:
- Ler `references/wfs.md` quando a tarefa envolver WFS, GetFeature, paginacao, outputFormat ou diferencas 1.0/1.1/2.0
- Ler `references/wms.md` quando envolver WMS, GetMap, GetFeatureInfo, diferencas 1.1.1 vs 1.3.0 ou BBOX
- Ler `references/wmts.md` quando envolver WMTS, tiles, TileMatrixSet, grid ou cache
- Ler `references/wcs.md` quando envolver WCS, coberturas raster, GetCoverage ou formatos de cobertura
- Ler `references/ogcapi.md` quando envolver OGC API Features, endpoints REST, colecoes, `items`, `limit` e `offset`
- Ler `references/crs.md` quando houver CRS/EPSG, ordem de eixo, transformacao ou `srsName/CRS`
- Ler `references/filters.md` quando envolver filtros OGC/FES, CQL/ECQL, operadores espaciais ou predicados
- Ler `references/errors.md` quando houver falhas de requisicao, `ExceptionReport`, erros de parametro ou diagnostico
