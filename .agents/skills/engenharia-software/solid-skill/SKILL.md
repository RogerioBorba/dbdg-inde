# Skill: SOLID

Use esta skill quando:
- A pergunta envolver qualquer principio SOLID
- Precisar revisar arquitetura orientada a objetos
- Refatorar classes com alta responsabilidade ou acoplamento
- Comparar alternativas de design com trade-offs claros

Instrucoes:
- Mapear o problema ao principio SOLID relevante
- Mostrar antes e depois da refatoracao de forma objetiva
- Explicar trade-offs tecnicos e impacto em manutencao
- Preferir solucoes simples e incrementalmente aplicaveis
