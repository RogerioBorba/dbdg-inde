# Skill: Code Review Expert

Use esta skill quando:
- Precisar revisar codigo com foco em defeitos reais e riscos
- Validar aderencia a SOLID, seguranca e qualidade de design
- Avaliar impacto de mudancas e cobertura de testes
- Sugerir correcoes objetivas com prioridade por severidade

Instrucoes:
- Priorizar findings por severidade com referencia de arquivo
- Explicar causa, risco e correcao recomendada de cada finding
- Evitar feedback generico sem evidencias no codigo
- Destacar lacunas de teste e riscos residuais
